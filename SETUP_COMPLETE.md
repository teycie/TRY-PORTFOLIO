## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd c:\Users\USER\Videos\TRYY
npm install
```

### 2. Start Development Server
```bash
npm start
```
The site will open at `http://localhost:3000`

### 3. Build for Production
```bash
npm run build
```
